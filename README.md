# restaurant-page

The Wizard’s Delights é um projeto front-end desenvolvido com HTML e CSS, inspirado no universo de Harry Potter. O site simula uma cafeteria mágica combinada com uma floricultura. 

A proposta do projeto é transportar o visitante para um ambiente fictício administrado por Dobby, onde cada detalhe — das cores aos textos — foi pensado para reforçar a identidade mágica do site. O foco principal esteve na organização do conteúdo, no uso de HTML e na construção de um layout utilizando apenas CSS, sem frameworks.

O site conta com uma página inicial temática, uma seção “Sobre Nós” com narrativa, um cardápio de doces mágicos, uma estufa de flores inspiradas nas casas de Hogwarts, avaliações fictícias feitas pelos protagonistas, integração com mapa para localização, uma seção interativa com mini-jogo de cupom de desconto e um rodapé com links para redes sociais.

Este projeto foi desenvolvido com fins educacionais e de portfólio, com o objetivo de praticar estruturação de páginas web, estilização com CSS puro, organização de layout e aplicação de criatividade no desenvolvimento front-end.
